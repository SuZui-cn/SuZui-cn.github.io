校园博客接口文档

账号模块

密码登录接口

URL:/api/login

请求方式：POST

请求参数

参数 类型 是否必须 描述
username String 是 用户名
password String 是 密码

响应参数

  • 登录成功
  • 登录失败

注册接口

URL:/api/register

请求方式：POST

请求参数

参数 类型 是否必须 描述
uid int 是 用户编号
upassword String 是 用户密码
uphone String 否 电话号码
uname String 是 用户名
usex String 否 性别
uemail String 否 邮箱

响应参数

  • 注册成功
  • 注册失败

邮箱登录接口

URL：/api/email

参数 类型 是否必须 描述
email String 是 邮箱
