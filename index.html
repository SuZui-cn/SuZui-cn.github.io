OpenEuler安装记录

0x01 在官网下载ios文件

下载地址： https://repo.openeuler.org/openEuler-21.09/ISO/x86_64/openEuler-21.09-x86_64-dvd.iso

0x02 在Vmware创建虚拟机

创建虚拟机时，请选择Linux类型和64-bit版本，20.09内核版本是4.19；如果选择安装21.03（内核版本为5.10），请选择Other Linux (64-bit)

为了在构建LFS系统时有较快的速度，内存设置为8GB（或以上）

0x03 系统安装

选择install openEuler 21.09

选择英文安装

此处需要填写root用户的密码

之后就开始自行安装

之后我就出问题了，显示boot load install failed ，这就很尴尬了。

问题是因为我下载的是debug版本的，下STL版本就可以正常使用了。

welcome to OpenEuler~
