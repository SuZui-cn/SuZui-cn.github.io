稀奇古怪的BUG

记录一下一位同学遇到问题，她的问题很复杂，很玄学，一度让我认为她的问题不是计算机科学的问题，而是哲学问题。

首先是安装MySQL的时候Microsoft yiswal C++ 2010 x64安装不成功，其次是jdk安装包打不开，但是jdk是老师发的，其他同学都可以正常安装，所以首先排除jdk安装包的问题。还有一点是谷歌的问题，谷歌问题倒是好解决，在Chrome官网下载最新的安装包安装之后切换搜索引擎就行了。

0x01 Microsoft yiswal C++ 2010 x64安装不成功

解决方法

下面给出几种可行的方案，但是在实际情况下可能会失效。

方法一：用修复软件修复

下载地址：https://support.microsoft.com/zh-cn/mats/program_install_and_uninstall

然而无果。

方法二：卸载显示为 Microsoft Visual C++ 2xxx Redistributable 的所有组件。

依然无果。

方法三：重置win系统

win + i打开设置

这里重置电脑的时候只重置C盘就行了。

0x02 JDK安装包打开时闪退

原因不明，暂时没有发现问题，试过jdk-8u191和jdk-8u161都没有成功。

网上说有一种情况是因为输入法问题，但是在切换为系统自带输入法的情况下依然没有效果。

最后安装32位的jdk成功。

pass：因为Oracle现在要注册账号才能下载jdk，所以这里给出腾讯的软件下载中心，https://pc.qq.com/search.html#!keyword=java

0x03 谷歌搜索无法使用

解决方法

切换谷歌搜索引擎
